std::vector<cv::KeyPoint> featureSet::keypointsList

List of keypoints associated with the image after min_thresh application. The Keypoint structure contains :

  • float angle : computed oriented of the keypoint(-1 if not applicable). The angle is in [0,360) degrees and measured relative to image coordinate system, i.e in clockwiwse
  • int class_id : object class, if the keypoints need to be clustered by an object they belong to
  • Point2d point : 2D coordinates of the keypoint in float
  • float response : the response by which the strongest keypoints have been selected
  • float size : diameter of the meaningful keypoint neighborhood
  • int octave : octave (pyramid layer) from which the keypoint has been extracted
